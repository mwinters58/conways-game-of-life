# 🔬 Conway's Game of Life

An interactive implementation of John Conway's famous cellular automaton - the Game of Life! Watch complex patterns emerge from simple rules.

## ✨ Features

- **Interactive Grid**: Click cells to toggle them alive/dead
- **Play/Pause Controls**: Watch evolution in real-time
- **Speed Control**: Adjust simulation speed (1-10x)
- **Step Mode**: Advance one generation at a time
- **Preset Patterns**: 
  - 🛸 Glider (moves across the grid)
  - 💫 Blinker (oscillates)
  - 🏮 Beacon (oscillates)
  - 🐸 Toad (oscillates)
  - 💥 Pulsar (complex oscillator)
  - 🔫 Gosper Glider Gun (generates infinite gliders!)
  - 🎲 Random (30% fill)
- **Statistics**: Live generation and population counters
- **Mobile Friendly**: Responsive design for all devices

## 🎮 How to Play

1. **Set Initial Pattern**: Click cells to make them alive (cyan) or use preset patterns
2. **Hit Play**: Watch the simulation evolve according to Conway's rules
3. **Experiment**: Try different patterns and speeds!

## 📐 The Rules

Conway's Game of Life follows four simple rules:

- **Birth**: Dead cell with exactly 3 neighbors becomes alive
- **Survival**: Live cell with 2-3 neighbors stays alive  
- **Death**: Live cell with <2 or >3 neighbors dies
- **Underpopulation/Overpopulation**: All other cases result in death
